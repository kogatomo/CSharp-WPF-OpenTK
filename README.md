# CSharp-WPF-OpenTK
C# + WPF(.NetFramework) + OpenTK

# 各プロジェクトの説明
1. キューブを表示する
2. キューブが回転するアニメーション
3. キーボードからの入力でキューブを動かす
4. マウスの操作でキューブを動かす

# ファイル
* MainWindow.xaml: 画面
* MainWindow.xaml.cs: 画面に対するソースコード

上記以外のファイルは設定ファイル。
